# Multi-Format Receipt Generator

A beginner Python project that takes input about item, and cost and delivers final receipt using functions, variables, format strings, and methods.
Put in upper-lower case jumbled names for item, yet you get a refined version in receipt. Question arepersonalized with an (s) for possibly plural items your purchase.

## Features
- Calculates totals
- Formats receipts neatly
- $ or ₹ symbol could be used in input without affecting the result.
- Uses Python functions, methods and f strings among others.
- Friendly code structure

## Technologies
- Python 3
